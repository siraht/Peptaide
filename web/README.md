# Peptaide Web

This directory contains the Next.js app.

Project overview, intent, and the full local-dev workflow live in `../README.md`.

## Quick Commands

From the repository root:

```bash
supabase start
```

From `web/`:

```bash
npm install
npm run dev
```

Quality gates:

```bash
npm run typecheck
npm test
npm run lint
npm run build
```
